# 旅行体験デモ

<!-- tip start -->

**このページについて**

このページは、LINE API Use Caseサイト（2026年3月31日に閉鎖）に掲載していた記事を、LINE Developersサイトへ移管したものです。なお、記事の内容は掲載時点のものです。

<!-- tip end -->

LINE上で実現するMaaSサービスなら、エンドユーザーの移動体験を、LINEミニアプリを使って全力でサポートできます。旅マエ（旅行前のシーン）、旅ナカ（旅行中のシーン）、旅アト（旅行後のシーン）、エンドユーザーのシチュエーションに合わせて適切な機能・システムを提供します。

旅マエでは、エンドユーザーが旅行先で使えるお得なチケットの中から自分に一番合ったものを選ぶことから始まり、予約、決済まで完結できます。また、旅行仲間の分までまとめて購入し、LINE上でスムーズにシェアすることも可能です。

旅ナカでは、旅行経路や旅行先の混雑状況を見える化することで、感染症対策を踏まえた安心・安全な移動体験をサポートできます。予定外の出来事や予定外の混雑状況などによってプラン通りに行動できなかったエンドユーザーに対しても、外部の地図サイトや自社の経路検索サイトと連携することで、次の目的地へのアクセス経路の再検索の手段を提供できます。

旅アトでのエンドユーザーの体験もサポート可能です。一度サービスを利用したユーザーに対して、お得な情報をLINEのメッセージで配信したり、そのメッセージを起点に次のサービスの予約や決済に繋げることが可能になります。

## 導入におけるメリット 

### エンドユーザー視点でのメリット 

#### 1. アプリのダウンロードや煩雑な入会手続きは不要 

サービスを利用するために、新たなアプリをダウンロードしたりインストールしたりする必要も、煩雑な入会手続きの必要もありません。さらに、チケット等はLINEのメッセージで受け取ることができるため、チケットを受け取るための複雑な手続きも必要ありません。

#### 2. 予約から支払いまでをLINE上で完結 

行き先（観光地など）を探すことから、予約、決済まで、すべてLIFFアプリで完結できます。

#### 3. 観光スポットの混雑状況が分かり、感染症対策にも 

感染症対策として、できる限り混雑した場所に行かないことが大切です。利用する公共交通機関や目的地の混雑状況を事前に確認できます。混雑を避けるためにプランを変更することも簡単にできます。

### サービス提供者視点でのメリット 

#### 1. 難しい手続きが必要なく、簡単に利用を開始してもらえます 

エンドユーザーが普段から慣れ親しんだLINEを通じて、新しい移動体験をスムーズに提供できます。

#### 2. 他ユーザーへの素早い拡散と浸透が期待できます 

多くのユーザーを抱えるLINEだからこそ、LINEを活用した極上の移動体験を提供し、エンドユーザーにメリットを感じてもらうことができれば、さらに多くのユーザーへの拡散・浸透も期待できます。

#### 3. エンドユーザーの操作履歴や来店履歴をもとにメッセージ配信可能です 

アプリでの操作履歴や来店履歴、購入履歴をもとに、新しい旅行先の提案や観光スポットでのレアなイベントの通知など、エンドユーザーにとって様々な情報をLINEのメッセージで配信できます。
